1. Project Objective

Build a real-world environment where an AI agent learns to:

Read customer support tickets
Classify issue type
Ask for missing information
Search knowledge base
Respond to customer
Resolve issue
Close ticket
Escalate when necessary
Optimize resolution time and customer satisfaction

This environment must simulate real workflows used by companies like SaaS companies, banks, e-commerce, telecom, etc.

This is NOT a game environment.
It must simulate real human work.

Reward must not be binary.
It must provide reward during the entire episode.

Example Reward Signals
Action	Reward

Searching knowledge base	+0.1
Asking relevant question	+0.15
Helpful response	+0.25
Resolving ticket	+0.4
Closing ticket	+0.2
Escalation	-0.2
Wrong resolution	-0.5
Infinite loop	-1.0
Too many steps	-0.2
Fast resolution	+0.2
High customer satisfaction	+0.3

Final reward must always be between 0.0 and 1.0.

Reward must consider:

time taken
number of steps
customer satisfaction
policy compliance
correct workflow order
